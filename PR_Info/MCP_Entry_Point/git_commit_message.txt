Add professional CLI command 'mcp-code-checker' with intelligent installation mode detection
and seamless fallback capabilities.

Key changes:

- Command detection and mode switching in integration.py
- CLI support methods added to ServerConfig class
- Automatic fallback: CLI command → Python module → Development mode
- Config tool integration for generating appropriate configurations
- Comprehensive test suite with cross-platform test scripts

The server can now be run with the clean CLI command:
  mcp-code-checker --project-dir /path/to/project

Instead of the previous Python module invocation:
  python -m src.main --project-dir /path/to/project

The config tool automatically detects and uses the best available command mode.
